## [1.0.0] - 2025-12-04

### Добавлено
- **Модуль 1**: Базовый каркас и конфигурация
  - Создание структуры проекта
  - Загрузка конфигурации из JSON
  - Настройка логирования

  - Поддержка сканированных PDF (через OCR с pdf2image и pytesseract)
  - Извлечение постранично с сохранением метаданных
  - Очистка данных (удаление HTML-тегов, навигационных элементов, управляющих символов)
  - Нормализация данных (расширение сокращений, стандартизация дат)
  - Структурирование текста (заголовки, абзацы, списки, FAQ-блоки)

- CLI интерфейс (squeezer.py)
- Поддержка кириллицы в PDF файлах
- Генерация отчётов и демонстраций трансформации текста

### Технические требования
- Python 3.8+
- pypdf
- pdf2image
- pytesseract
- Pillow
- chardet
