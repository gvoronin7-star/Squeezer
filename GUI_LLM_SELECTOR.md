# GUI: Селектор LLM моделей

## Обновление интерфейса

Добавлен улучшенный селектор моделей LLM в GUI с пояснениями для каждой модели.

---

## 📸 Интерфейс

```
┌─────────────────────────────────────────────────────────────┐
│ Настройки                                                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ Модель LLM: [GPT-4o Mini - Для обычной обработки ▼]        │
│             ⚡ Быстрая и дешёвая                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

---

## 🎯 Доступные модели

| # | Название | Use Case | Описание | Model ID |
|---|----------|----------|----------|----------|
| 1 | **GPT-4o Mini** | Для обычной обработки | ⚡ Быстрая и дешёвая | `gpt-4o-mini` |
| 2 | **GPT-4o** | Для важных задач | ⭐ Лучшее качество OpenAI | `gpt-4o` |
| 3 | **Claude Sonnet 4.6** | Рекомендуется | 🏆 Лучший баланс (1M контекст) | `claude-sonnet-4-6` |
| 4 | **Claude Haiku 4.5** | Для экономии | 💨 Быстрый Claude | `claude-haiku-4-5` |
| 5 | **Claude Opus 4.6** | Для сложных задач | 👑 Максимальное качество | `claude-opus-4-6` |

---

## 💡 Рекомендации по выбору

### По умолчанию: **GPT-4o Mini**
- ⚡ Самая быстрая
- 💰 Самая дешёвая
- ✅ Работает через OpenAI SDK
- 📝 Идеально для обычной обработки

### Для качества: **Claude Sonnet 4.6**
- 🏆 Лучший баланс цена/качество
- 📚 **1M токенов контекста** - для больших документов
- ✅ Работает через Anthropic SDK
- 🌍 Хорошая поддержка русского языка

### Для экономии: **Claude Haiku 4.5**
- 💨 Самый быстрый Claude
- 💰 Дешёвый
- ✅ Работает через Anthropic SDK

### Для сложных задач: **Claude Opus 4.6**
- 👑 Максимальное качество
- 🧠 Для сложных задач анализа
- ✅ Работает через Anthropic SDK

## 🔧 Технические детали

### Формат данных в коде:

```python
LLM_MODELS = [
    (model_id, display_name, description, use_case),
    ...
]
```

### Пример:

```python
LLM_MODELS = [
    ("gpt-4o-mini", "GPT-4o Mini", "⚡ Быстрая и дешёвая", "Для обычной обработки"),
    ("claude-sonnet-4-6", "Claude Sonnet 4.6", "🏆 Лучший баланс (1M контекст)", "Рекомендуется"),
]
```

## 🎨 Особенности GUI

1. **Выпадающий список** с понятными названиями
2. **Описание под списком** обновляется при выборе модели
3. **Иконки** для визуального различия:
   - ⚡ - скорость
   - ⭐ - качество
   - 🏆 - рекомендуется
   - 💨 - экономия
   - 👑 - премиум
