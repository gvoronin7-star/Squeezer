## 1. Обзор проекта

**Соковыжималка (Squeezer)** — это система обработки PDF документов с поддержкой RAG (Retrieval Augmented Generation).

### Назначение

- Извлечение текста из PDF с поддержкой OCR
- Разбиение на семантические чанки
- Векторизация и сохранение в FAISS
- Создание готовых RAG-баз данных
- Улучшенный поиск с Re-ranking, HyDE, Fusion
- Self-RAG с самооценкой качества
- Генерация ответов с цитатами

### Ключевые технологии

- Python 3.8+
- OpenAI API (embeddings + LLM)
- FAISS (векторная БД)
- Tkinter (GUI)
- pdfplumber (таблицы)
- Anthropic Claude (опционально)

## 5. Технические детали

### Обработка PDF

| Параметр | Значение |
|----------|----------|
| Извлечение | pypdf + pdfplumber |
| OCR | Tesseract (опционально) |
| Таблицы | pdfplumber + LLM |
| Языки | RU + EN (автоопределение) |

### Чанкинг

| Параметр | Значение по умолчанию |
|----------|----------------------|
| max_chunk_size | 500 символов |
| overlap | 50 символов |
| Типы блоков | heading, paragraph, list |

### Векторизация

| Параметр | Значение |
|----------|----------|
| Модель | text-embedding-3-small |
| Размерность | 1536 |
| Векторная БД | FAISS (IndexFlatL2) |
| Кэш | 100MB, 30 дней |

### RAG Метрики

| Техника | Улучшение |
|---------|-----------|
| Re-ranking | +15-30% precision |
| HyDE | +10-20% recall |
| Fusion | +20-40% combined |
| Query rewriting | +20-40% recall |
| Self-RAG | +качество ответов |
